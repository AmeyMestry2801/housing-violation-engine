How it works

Click and drag on the image to create a region. Then assign a violation to that region and save it. You can repeat this for multiple violations in the same image.

While zoomed in, use Pan Mode or hold Space and drag to move around the image.

Current demo supports JPG/PNG uploads. HEIC support would be added later through image conversion.

For Bad Roof, focus on holes, tarps, and visible discoloration.
